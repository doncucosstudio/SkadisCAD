SkadisCAD Help & Guide

↖
Select & Move
Click the Select tool then click any shape to select it. Drag to reposition. Use arrow keys ↑↓←→ to nudge one 40mm grid step at a time.
📐
Board size
Pick a standard IKEA Skadis size from the Board dropdown in the toolbar, or choose Custom… and enter exact mm dimensions.
🔍
Zoom & Snap
Change zoom with the Scale dropdown. Snap controls whether shapes align to the 40mm hole grid, half-grid (20mm), or move freely.
🎨
Fill color
Select a shape then click a swatch in the Fill Color panel (right side) to color-code different regions.
Tip: Ctrl+Z undoes the last action. Del removes the selected shape. Esc deselects. Ctrl+A selects the topmost shape.
⬡
Preset shapes
Click any preset in the left panel to instantly add it centered on the board. Presets include full boards, rectangles, strips, circles, ovals, hexagons, L-shapes, T-slots, and arches.
▭
Draw tools
Choose Rectangle, Ellipse, or Polygon from the Tools section, then drag on the canvas to draw. The shape snaps to the current snap mode.
📏
Custom dimensions
Enter exact mm values in the Custom Rectangle or Custom Ellipse sections and click Add to place a precisely sized shape.
📋
Properties
Select a shape and edit its X, Y, width, and height in the Properties panel on the right. Values snap to the grid step you enter.
Tip: Shapes are stacked in the order shown in the Shapes list. The topmost shape in the list is hit-tested last (appears on top visually).
⊕
Center holes on shape
Resets the hole grid origin to the exact center of the selected shape. The grid is always anchored to the center and radiates outward — this snaps it back after any offset.
⬡
Stagger rows
Toggle on to offset every other row by half a grid step (20mm at 40mm pitch), creating a honeycomb pattern. Hole size and spacing remain IKEA-spec compliant.
±
Hole count — fewer / more
The − fewer button insets the placement boundary by 20mm so outer holes disappear. + more extends the boundary by 20mm so additional holes appear near the edges. Hole size and 40mm pitch never change. A dashed amber rectangle shows the active boundary.
↔
Offset X / Y
Shifts the entire hole grid relative to the shape center without moving the shape. Use the nudge buttons to move by one full grid step, or type a precise mm value. An orange crosshair marks the shifted grid origin.
📐
Grid step
Changes the pitch between holes for that shape only. Default is 40mm (IKEA standard). Increase for sparser layouts, decrease for denser ones. Hole size stays at 5×15mm.
Tip: All hole controls are per-shape — each shape on the board can have its own independent stagger, offset, count, and grid step settings.
📐
SVG
Best for laser cutters (Lightburn, RDWorks), Cricut, Silhouette, and direct browser use. Fully scalable, includes exact pill-shaped holes with correct mm dimensions in the viewBox.
📏
DXF
Standard format for AutoCAD, Fusion 360, VCarve, Carbide Create, and most CAM software. Holes are exported as precise closed polylines. Layers: OUTLINE (shape boundary), HOLES (pill cutouts). Compatible with AutoCAD, Fusion 360, VCarve, and all major CAM software.
⚙
G-Code (.nc)
Ready-to-run CNC toolpath. Configured for 5mm cut depth, 1200mm/min feed, 300mm/min plunge, 18000 RPM spindle. Edit the header constants to match your machine and material.
🖨
PDF Template
Opens a print dialog with the design scaled to exact 1:1 mm dimensions. Use this to print a drilling template on paper and transfer hole positions manually.
🖼
PNG Preview
Exports a 4× high-resolution raster image of the board. Useful for documentation, sharing, or previewing the layout. Not suitable for cutting — use SVG or DXF for that.
Tip: For laser cutting, use SVG. For CNC routing with CAM software, use DXF. For direct machine control without CAM, use G-Code.
